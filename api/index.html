s2fft.logs.setup_logging(custom_yaml_path=None, default_level=10)

initialise and configure logging.

Should be called at the beginning of code to initialise and configure the desired logging level. Logging levels can be ints in [0,50] where 10 is debug logging and 50 is critical logging.

Parameters

  • custom_yaml_path (string) – Complete pathname of desired yaml logging configuration. If empty will provide default logging config.

  • default_level (int) – Logging level at which to configure.

Raises

ValueError – Raised if logging.yaml is not in ./logs/ directory.

s2fft.wigner.trapani.compute_eighth(dl: numpy.ndarray, L: int, el: int) → numpy.ndarray

Compute Wigner-d at argument \(\pi/2\) for eighth of plane using Trapani & Navaza recursion.

The Wigner-d plane is computed by recursion over \(\ell\) (el). Thus, for \(\ell > 0\) the plane must be computed already for \(\ell - 1\). For \(\ell = 1\) the recusion must already be initialised (see init()).

The Wigner-d plane \(d^\ell_{mm^\prime}(\pi/2)\) (el) is indexed for \(-L < m, m^\prime < L\) by dl[m + L - 1, m’ + L - 1] but is only computed for the eighth of the plane \(m^\prime <= m < \ell, 0 <= m^\prime <= \ell\). Symmetry relations can be used to fill in the remainder of the plane if required (see fill_eighth2quarter(), fill_quarter2half(), fill_half2full()).

Warning

This recursion may not be stable above \(\ell \gtrsim 1024\).

Parameters

  • dl – Wigner-d plane for \(\ell - 1\) at \(\pi/2\). If \(\ell = 0\) the recursion is initialised internally and the dl argument is ignored.

  • L – Harmonic band-limit.

  • el – Spherical harmonic degree \(\ell\).

Returns

Plane of Wigner-d for el, with eighth of plane computed.

s2fft.wigner.trapani.compute_full(dl: numpy.ndarray, L: int, el: int) → numpy.ndarray

Compute Wigner-d at argument \(\pi/2\) for full plane using Trapani & Navaza recursion.

The Wigner-d plane is computed by recursion over \(\ell\) (el). Thus, for \(\ell > 0\) the plane must be computed already for \(\ell - 1\). For \(\ell = 1\) the recusion must already be initialised (see init()).

The Wigner-d plane \(d^\ell_{mm^\prime}(\pi/2)\) (el) is indexed for \(-L < m, m^\prime < L\) by dl[m + L - 1, m’ + L - 1]. The plane is computed directly for the eighth of the plane \(m^\prime <= m <0 \ell, 0 <= m^\prime <= \ell\) (see compute_eighth()). Symmetry relations are then used to fill in the remainder of the plane (see fill_eighth2quarter(), fill_quarter2half(), fill_half2full()).

Warning

This recursion may not be stable above \(\ell \gtrsim 1024\).

Parameters

dl – Wigner-d plane for \(\ell - 1\) at \(\pi/2\). If

:param \(\ell = 0\) the recursion is initialised internally and the dl: :param argument is ignored.: :param L: Harmonic band-limit. :param el: Spherical harmonic degree \(\ell\).

Returns

Plane of Wigner-d for el, with eighth of plane computed.

s2fft.wigner.trapani.fill_eighth2quarter(dl: numpy.ndarray, L: int, el: int) → numpy.ndarray

Fill in quarter of Wigner-d plane from eighth.

The Wigner-d plane passed as an argument should be computed for the eighth of the plane \(m^\prime <= m <0 \ell, 0 <= m^\prime <= \ell\). The returned plane is computed by symmetry for \(0 <= m, m^\prime <= \ell\).

Parameters

  • dl – Eighth of Wigner-d plane for \(\ell\) at \(\pi/2\).

  • L – Harmonic band-limit.

  • el – Spherical harmonic degree \(\ell\).

Returns

Plane of Wigner-d for el, with quarter of plane computed.

s2fft.wigner.trapani.fill_half2full(dl: numpy.ndarray, L: int, el: int) → numpy.ndarray

Fill in full Wigner-d plane from half.

The Wigner-d plane passed as an argument should be computed for the half of the plane \(-\ell <= m <= \ell, 0 <= m^\prime <= \ell\). The returned plane is computed by symmetry for \(-\ell <= m, m^\prime <= \ell\).

Parameters

  • dl – Quarter of Wigner-d plane for \(\ell\) at \(\pi/2\).

  • L – Harmonic band-limit.

  • el – Spherical harmonic degree \(\ell\).

Returns

Plane of Wigner-d for el, with full plane computed.
